自作言語入力インターフェース

これはスタック型のプログラミング言語で操作できる、JavaScriptプレイグラウンドです。

1

数値

数値をスタックに積む

10 20 30

2

四則演算 + - * /

上2つを取り出して計算し、結果を積む

3 4 + → 7

3

比較 = < >

上2つを比較し、真なら1・偽なら0を積む

3 5 < → 1

4

print

トップの値をコンソールに出力する

42 print

5

place

x y w h で箱を生成し、IDを積む

50 50 80 80 place

6

move

x y id で箱を絶対座標へ移動

200 100 id move

7

rmove

dx dy id で箱を相対移動

50 0 id rmove

8

resize

w h id で箱のサイズを変更

120 120 id resize

9

color

r g b id で箱の色を変更

255 0 0 id color

10

opacity

val id で透明度を変更（0〜1、小数は割り算で）

1 2 / id opacity → 0.5

11

delete

id の箱を爆発エフェクトで削除

id delete

12

fade

duration id で箱をフェードアウト消去

500 id fade

13

clone

id の箱を同じ位置に複製し、新IDを積む

id clone

14

cloneAt

x y id で指定位置に複製し、新IDを積む

300 200 id cloneAt

15

spin

id の箱を回転アニメーション

id spin

16

shake

id の箱を振動アニメーション

id shake

17

loop

n [ ... ] でブロックをn回繰り返す

5 [ id spin ] loop

18

if

cond [ ... ] で条件が真ならブロック実行

1 [ id delete ] if

19

random

n を取り出し、0〜nの整数乱数を積む

100 random

20

sleep

ms ミリ秒だけ処理を待機する

500 sleep

21

変数定義 ->

トップの値を変数に保存する

42 -> x

22

変数参照

変数を呼び出す（中身がブロックなら実行）

x

23

dup

トップを複製する　[a] → [a a]

5 dup

24

swap

上2つを交換する　[a b] → [b a]

1 2 swap

25

over

2番目を複製して積む　[a b] → [a b a]

1 2 over

26

rot

上3つを回転する　[a b c] → [b c a]

1 2 3 rot

27

ブロック [ ]

命令の並びを値としてまとめて積む

[ 1 2 + ]
